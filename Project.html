Projects

1. Cyber Security Based SIEM Monitoring System Using Splunk

It is a security-based project where we use the Splunk tool to store and safely monitor data in real-time. Splunk is connected with Apache Server, Splunk Server, and Fortigate Server to ensure real-time data monitoring and protect data from threats. Alerts are sent via email to the client if any abnormal events occur during monitoring.

2. Library Management System

This is a DBMS project that involves the management of books in a library using PHP for the backend and HTML, CSS, and JavaScript for the frontend. It runs on an Apache Server with MySQL Server.

3. Bubble Fish Game App

This is a game app created using Java and XML languages, running on Android Studio. It involves players using flippers to hit a ball around a table, aiming to score points by hitting various targets and obstacles.
